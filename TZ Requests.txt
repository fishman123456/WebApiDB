Задание: реализовать приложение, работающее с БД, которое регистрирует информацю о выполненных к нему запросах и записывает ее в БД.

Стек технологий: C#/ASP.NET/EF + MS SQL Server/SSMS
Тип приложения: JSON Web-API

Поддерживаемые запросы:

GET / -> {
	"message": "server is running", 
	"time": <now time>
}

GET /ping -> {
	"message": "pong",
	"time: <now time>
}

GET /requests -> [
	<массив записей из таблицы с информацией о сделанных запросах>
]

GET /requests/<id> -> {
	<информацию о запросе с заданным id>
}

При выполнении каждого запроса к API необходимо добавлять в БД запись об этом запросе. Информацию для записи определить самостоятельно.
Примеры данных, которые можно записать в БД: url, request method, response status code, ...

Реализовать приложение, протестировать на локальной БД. 
Затем использовать в приложении облачную БД на https://freeasphosting.net/. Протестировать работу с облачной БД.
